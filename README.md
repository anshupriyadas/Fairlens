# FairLens — Contextual Bias Intelligence Platform

## 🚀 Overview
FairLens is an AI-powered platform that detects, explains, and simulates bias in machine learning models.

## 💡 Features
- Bias Detection
- Decision Archaeology (Explainability)
- Counterfactual Sandbox
- Harm Probability Score
- Bias Heatmaps
- AI Explanation Chatbot

## 🧠 How it works
1. Upload dataset + model
2. Detect bias
3. Explain why it happens
4. Simulate alternative outcomes
5. Generate risk score and report

## 🛠 Tech Stack
- React + Tailwind
- FastAPI (Python)
- SHAP + DiCE-ML
- PostgreSQL
- Google Gemini API

## 🎯 Use Case
Ensuring fairness in:
- Loan approvals
- Hiring systems
- Healthcare AI
